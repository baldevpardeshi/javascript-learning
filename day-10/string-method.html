String Method

  1. Length:
    use to count length of string
    const a = "string"
    a.length
  2. charAt:
    Use to take a any character or digit in the String
    const a = "string"
    a.charAt[0]
  3. slice:
    use to slice a text from the string
    const names = "apple,banana,kiwi"
    names.slice(6, 12)
  4. subString:
    Use to create another separate string from the string
    const names = "apple,banana,kiwi"
    names.slice(6, 6)
    We can also use in (-4) t0 start it from the end of the string
  5. toUppercase:
    use to transform string to capital
    const a = "string"
    toUppercase(a)
  6. toLowercase:
    use to transform string to small case
    const a = "string"
    toLowercase(a)
  7. Concat
    Use to concat(join) two and more strings.
    const a = "string"
    const b = "string"
    a.concat(" ", b)
  8. trim()
    use to remove leading and trailing whitespace.
    const a = " string "
    a.trim()
    also have two more options: trimStart and trimEnd, to remove leading and trailing whitespace.
  9. padStart
    Use to add some extra same text into the output text.
    const a = "5987"
    a.padStart(13, "x")
    output: xxxx xxxx xxxx 5987
  10. Replace:
    use to replace the text in the output text.
    const a = "hello world"
    a.replace("l", "*");
    output: he*lo;
  11. Replace All:
    use to replace the text in the output text.
    const a = "hello world"
    a.replaceAll("o", "*");
    output: hell* w*rld;
  12. Split: use to separate the string usint the " ";
    const names = "apple,banana,kiwi"
    names.split("a")
    output: ["","pple,b", "n", "n", ",kiwi"];
