datatypes:

String:

Example: let str = "Hello, World!"; Explanation: Strings are used to represent text. They are enclosed in either single or double quotes.

Number:

Example: let num = 42; Explanation: Numbers can represent integers or floating-point values. They are not enclosed in quotes.

Boolean:

Example: let isTrue = true; Example: let isFalse = false; Explanation: Booleans represent true or false values. They are not enclosed in quotes.

BigInt:

Example: let bigNumber = 1234567890123456789012345678901234567890n; Explanation: BigInt is used to represent very large integer values. It is suffixed with the letter "n" to indicate it's a BigInt.

Object:

Example: let obj = {}; Explanation: Objects are used to store key-value pairs and are enclosed in curly braces.

Array:

Example: let arr = [1, 2, 3]; Explanation: Arrays are ordered lists that can contain values of different data types and are enclosed in square brackets.

Undefined:

Example: let a; Explanation: Variables that are declared but not initialized have the value undefined.

Null:

Example: let a = null; Explanation: Null is a special value that represents the intentional absence of any object value.

Symbol:

Example: const uniqueSymbol = Symbol('description'); Explanation: Symbols are unique and immutable values often used as object property keys to ensure their uniqueness.
