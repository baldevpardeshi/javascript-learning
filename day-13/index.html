Array Method

  • Flat()
    use to convert array of arrays to one array.
  • slice()
    use to slice one or more index from the array.
    splice method slices out pices of an array.
    slice method does not remove any element from the source array.
    if you want to remove multiple array index you can use slice(1,3) it will remove the 1 and 2 indexes.
  • splice()
    The splice method add new items to the source array.
