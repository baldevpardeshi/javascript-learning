Array Method

  • lenght:
    use to count number in the array;
  • toString()
    use to convert the array to a string;
  • at()
    use to select array index;
  • join() use to join the array into single string with the given separator
  • push()
    use to push new value into created array at the last
  • unShift()
    use to push new value into created array at the start
  • pop()
    use to remove last value from created array
  • shift() use to remove first value from created array
  • delete()
    use to remove the value but not change the value of the lenght
